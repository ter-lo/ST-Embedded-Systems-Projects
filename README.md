# Configure GPIO Addresses from Scratch
Board used: STM32-F030R8

The goal of this project is to turn on the user LED, with the sub goal of toggling the user LED with the user button.

Documentation for STM32-F030R8:
- User Manual, https://www.st.com/resource/en/user_manual/um1724-stm32-nucleo64-boards-mb1136-stmicroelectronics.pdf
- Reference Manual, https://www.st.com/resource/en/reference_manual/rm0360-stm32f030x4x6x8xc-and-stm32f070x6xb-advanced-armbased-32bit-mcus-stmicroelectronics.pdf
- Datasheet, https://www.st.com/resource/en/datasheet/stm32f030f4.pdf
